# Mercado Pago Integration App

> This app has a purpose to show a simple integration with Mercardo Pago API Service

## Features

- [x] Add Amount
- [x] Select Pay Method
- [x] Select Bank
- [x] Select Installments
- [x] Summary
- [x] Finish payment
